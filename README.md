# 📝 ToDo App

A simple and user-friendly ToDo application that helps you stay organized by keeping track of your daily tasks.

## 🚀 Features

- ✅ Add new tasks
- 🗑️ Delete tasks
- 📌 Mark tasks as completed
- 🎨 Clean and intuitive user interface
